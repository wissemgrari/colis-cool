
# Colis Cool

Colis Cool est une plateforme de livraison collaborative entre voisins.


## Installation

Install colis cool with npm.

```bash
  git clone https://github.com/wissemgrari/colis-cool.git
  cd colis-cool
  npm install
```
    
## Deployment

To deploy this project run

```bash
  yarn start
  or
  npm start
```
